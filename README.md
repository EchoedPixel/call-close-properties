# Call & Close Properties

Premium real estate lead generation, appointment setting, and sales support website.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Brand Configuration

Update company details in `lib/constants.ts`:

- Company name, email, phone
- Google Calendar booking URL (`bookingUrl`)
- Social links
- Services, testimonials, team
